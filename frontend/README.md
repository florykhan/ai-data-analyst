# Frontend (React / Next.js)

Dataset upload, dashboards, chat UI, and visualizations for AI DataLab.

## Structure

- `components/` — Reusable UI components
- `pages/` — Page-level views
- `charts/` — Chart components (Plotly, Recharts)
- `chat/` — Conversational AI interface

## Stack

- React / Next.js
- TailwindCSS
- Plotly, Recharts

## Setup

```bash
npm install
# or: pnpm install / yarn install
```

## Run

```bash
npm run dev
```

*(Initialize with `npx create-next-app` or similar when starting implementation.)*
